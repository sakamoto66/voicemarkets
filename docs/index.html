VoiceMarkets

声でブックマーク・履歴を検索する Chrome 拡張機能

GitHub でインストール 使い方を見る

VoiceMarkets とは

ブックマークが何千件もあって探すのが大変になっていませんか？ VoiceMarkets はマイクボタンをクリックして話しかけるだけで、あなたのブックマークや閲覧履歴からピッタリのページを即座に見つけます。 外部 API 不要 — データはブラウザの外に出ません。

特徴

声で検索

マイクボタンを押して話すだけ。キーボード入力不要。

Gemini Nano AI ランキング

Chrome に内蔵された Gemini Nano がオンデバイスでセマンティック検索。

プライバシー重視

ブックマーク・履歴データは一切外部に送信されません。

日本語対応

日本語・英語のバイリンガルキーワード抽出に対応。

フォールバック

AI 非対応環境でもキーワード検索として動作します。

期間・ソースフィルター

1時間〜全期間、ブックマーク/履歴を切り替えて絞り込み。

インストール

  1. GitHub からリポジトリをダウンロードまたはクローン
  2. chrome://extensions/ を開きデベロッパーモードを有効化
  3. 「パッケージ化されていない拡張機能を読み込む」でフォルダを選択
  4. ツールバーのアイコンをクリックして開始
