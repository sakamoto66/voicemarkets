プライバシーポリシー（日本語）

VoiceMarkets（以下「本拡張機能」）は、あなたのブックマークおよびブラウザ履歴を音声で検索するための Chrome 拡張機能です。 本ポリシーは、本拡張機能がどのようにデータを扱うかを説明します。

収集・送信するデータ

本拡張機能は、あなたのブックマークや閲覧履歴を外部サーバーに送信しません。 これらのデータはすべてブラウザ内で処理されます。

音声認識には Chrome の Web Speech API を使用しています。 音声データは Google のサーバーに送信されて文字起こしされますが、これは Chrome ブラウザ標準の音声入力と同じ仕組みです。 本拡張機能自身のサーバーには何も送信されません。

Gemini Nano によるセマンティックランキングは、デバイス上で完結します。 候補ページのタイトル・URL・音声テキストが Gemini Nano モデルに渡されますが、このモデルは Chrome に組み込まれており、外部ネットワーク通信は行われません。

第三者への提供

本拡張機能は、あなたのデータを第三者に販売・共有・提供することはありません。

お問い合わせ

ご質問は GitHub Issues までお寄せください。

Privacy Policy (English)

VoiceMarkets is a Chrome extension that lets you search your bookmarks and browser history by voice. This policy explains how the extension handles your data.

Data collected and transmitted

VoiceMarkets does not transmit your bookmarks or browsing history to any external server. All data is processed locally within your browser.

Voice recognition uses Chrome's Web Speech API. Your voice audio is sent to Google's servers for transcription — this is the same mechanism used by Chrome's built-in speech input. No audio or transcription data is sent to any server operated by this extension.

Semantic ranking via Gemini Nano runs entirely on-device. Candidate page titles, URLs, and the transcribed voice query are passed to the Gemini Nano model built into Chrome. No network request is made for this step.

Third parties

This extension does not sell, share, or disclose your data to any third party.
